# 🚀 Frontend Assessment Project

A modern and responsive frontend application built using **React + Vite + TypeScript + Tailwind CSS**.  
This project was developed as part of a frontend assessment to demonstrate UI development, component architecture, and responsive design skills.

## 📌 Features

- ⚡ Fast development with Vite  
- 🧠 Type-safe code using TypeScript  
- 🎨 Modern UI with Tailwind CSS  
- 📱 Fully responsive design (mobile + desktop)  
- 🧩 Reusable component-based architecture  
- 🖼️ Asset handling via `public`  
- 🚀 Deployed on Vercel with auto CI/CD  

## 📁 Project Structure
public/ # Contains static images and assets
src/
├── api/ # Dummy API logic and services
├── components/ # Reusable UI components
├── data/ # Static / dummy data files
├── types/ # TypeScript type definitions
├── App.tsx # Root component
└── main.tsx # Entry point




⚙️ Setup Instructions
1. Clone repository
git clone https://github.com/rbgraja/Frontend-Assesment/
2. Navigate to project
cd frontend
3. Install dependencies
npm install
4. Run development server
npm run dev
5. Open in browser
http://localhost:5173


📦 Build for Production
npm run build
